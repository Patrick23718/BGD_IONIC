Questions fréquentes

PROBLÈME DE PAIEMENT AVEC MA CARTE ?
Il peut y avoir plusieurs explications : Avez-vous vérifié la date de validité de votre carte de paiement visible sur sa face avant ?
Avez-vous vérifié le numéro de carte que vous avez saisi, le code de sécurité? (les 3 derniers numéros au dos de la carte) 
Avez-vous bien saisi votre nom tel qu'affiché sur votre carte? 
Si vous disposez d'une nouvelle carte, il est possible que votre banque vous demande de faire un premier retrait avant de pouvoir l'utiliser sur internet.
Peut-être avez-vous atteint votre plafond de paiement. Dans ce cas, nous vous invitons à demander à votre banque de le relever.
3D SECURE+

Il arrive parfois qu'un contrôle de la transaction soit réalisé par votre banque à l'aide du 3D Secure. Lorsque cela arrive, votre banque affiche une fenêtre de vérification permettant le contrôle du 3D Secure. En cas de problème de chargement ou d'affichage, nous n'avons malheureusement pas intervenir.

Si vous ne recevez pas le SMS de vérification, assurez-vous que votre banque l'adresse bien au bon numéro de téléphone. 

AJOUTER UN CODE DE RÉDUCTION ?
Vous avez un code de réduction et vous aimeriez l’utiliser pour votre prochain rendez-vous ? Il suffit de réserver la prestation de votre choix sur notre plateforme.  Une fois arrivé sur la page du paiement, vous trouverez une mention "j'ai un code promo"
VOUS SOUHAITEZ ANNULER VOTRE RÉSERVATION ?
Vous avez la possibilité d'annuler gratuitement votre réservation jusqu'à 24h avant l'heure du rendez-vous. Il vous suffit pour cela de nous contacter en nous envoyons un message depuis NOUS CONTACTER dans votre compte.
Après annulation de votre réservation, vous recevrez votre remboursement entre 3 et 5 jours.
QUAND AVEZ-VOUS DROIT À UN REMBOURSEMENT ?
Nous vous permettons d'annuler votre réservation jusqu'à 24h avant votre rendez-vous.  Par ailleurs, un remboursement est possible dans les conditions suivantes :
La coiffeuse a annulée ou a un empêchement de dernière minute.
Si la coiffeuse a annulé votre rendez-vous à l'avance et par téléphone / SMS.
Si vous avez annulée 24h avant le début de la prestation.
METTRE MES COIFFEUSES PRÉFÉRÉES EN FAVORIS :
Pour mettre vos coiffeuses préférées en favoris, il vous suffit de cliquer sur le cœur présent sur le profil de la coiffeuse. Et pour les retrouver dans votre compte, il vous suffit de vous rendre dans votre compte, puis de cliquer sur paramètres puis sur mes coiffeuses préférées.
